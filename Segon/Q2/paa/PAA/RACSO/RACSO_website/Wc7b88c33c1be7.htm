Exercise ‹2›:

Surrounding numbers
Note: this problem has been taken from the Logic in Information Technology subject at Barcelona School of Informatics.

Solve the following exercise by means of a reduction to SAT:
  • Given an n×mn\times mn×m board such that some of the cells contain a number in {0,…,3}\{0,\ldots,3\}{0,…,3}, find a way to draw a set of cycles such that, given a cell with a number kkk, exactly kkk of its four sides are part of a cycle. Cycles can’t share sides nor vertices. The sides of a cell can belong to different cycles. The input of the exercise and the output with the solution (when the input is solvable) are as follows: 
  • board: array of array of int
    The input is an n×mn\times mn×m matrix. Each of its cells either has a value in {0,…,3}\{0,\ldots,3\}{0,…,3}, or it has the value −1-1−1 to denote that the cell is empty. 
  • segments: array of array [2] of struct {
   r: int
   c: int
}
    The output is a list of pairs of adjacent cells. Each cell is identified with its coordinates r (row) and c (column). Each of the pairs of adjacent cells denotes that the segment adjacent to both cells is part of a cycle.

    Note that in some cases it will be necessary to identify cells that are outside the board (see, e.g., the small cycle surrounding two 333-cells in the example, which involves four segments that separate a cell from the void outside the board). For this reason, the value of r can be any number in {−1,…,n}\{-1,\ldots,n\}{−1,…,n} instead of {0,…,n−1}\{0,\ldots,n-1\}{0,…,n−1}, and the value of c any number in {−1,…,m}\{-1,\ldots,m\}{−1,…,m} instead of {0,…,m−1}\{0,\ldots,m-1\}{0,…,m−1}: this allows to identify one extra cell around the whole board.
