Worker selection
Solve the following exercise by means of a reduction to SAT:
  • A company must complete mmm different tasks. There are nnn workers, each of which can perform a subset of the tasks. We want to find a set of exactly kkk workers, with k<nk<nk<n, such that every task can be performed by at least one of the workers of the set.
The input of the exercise and the output with the solution (when the input is solvable) are as follows: 
  • n: int
m: int
k: int
tasks: array of array of int
    The input consists of the naturals nnn, mmm, kkk, and for each task the list indicating which workers can perform it in tasks. The position (i,j)(i,j)(i,j) of tasks contains a 111 if the worker jjj can perform the task iii, or 000 otherwise. The tasks are identified by numbers between 000 and m−1m-1m−1. Similarly, the workers are identified by numbers between 000 and n−1n-1n−1. 
  • set: array of int
    The output is a list containing the identifiers of the workers that are part of the set.
