Exercise ‹5›:

Sequence matching
Solve the following exercise by means of a reduction to SAT:
  • Given an n×nn\times nn×n board MMM of 000’s and 111’s and an n2n^2n2-long sequence SSS of 000’s and 111’s, determine if it is possible to find a path through adjacent cells of MMM, starting at its top-left cell and going through each of its cells exactly once, matching the sequence SSS.
The input of the exercise and the output with the solution (when the input is solvable) are as follows: 
  • M: array of array of int
S: array of int
    The input contains the board as a matrix MMM with nnn rows and nnn columns of 000’s and 111’s, and the sequence as an array SSS with n2n^2n2 000’s and 111’s. The top-left cell is the cell (0,0)(0,0)(0,0). 
  • path: array of array of int
    The output contains the details of the path. It is a two-dimensional matrix that must have size n×nn\times nn×n, with the position (i,j)(i,j)(i,j) indicating the index in the path of the cell (i,j)(i,j)(i,j), i.e., a value between 000 and n2−1n^2-1n2−1. Thus, the position (0,0)(0,0)(0,0) of path must always contain a 000, since the path starts at the cell (0,0)(0,0)(0,0).
